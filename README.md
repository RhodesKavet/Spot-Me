# SpotMe 🏋️

> Find your people. Find your gym. Get after it.

SpotMe is a fitness social platform where athletes connect through shared gym locations, post workouts, and compete on leaderboards.

## Features
- 🔐 Real user authentication (sign up / sign in)
- 📱 Scrolling social feed with posts, likes, and tags
- 🗺️ Interactive worldwide gym map — see who trains where
- 🤝 Follow users, connect with gym members nearby
- 🏆 Community leaderboard
- 💪 Fitness challenges
- 🔍 Explore content by category

## Tech Stack
- **Frontend:** Vanilla HTML/CSS/JS
- **Backend/Auth/DB:** Supabase
- **Hosting:** Vercel
- **Map:** Leaflet.js + OpenStreetMap

## Setup

1. Clone this repo
2. Run the SQL in `schema.sql` in your Supabase SQL Editor
3. Deploy to Vercel by importing this repo
